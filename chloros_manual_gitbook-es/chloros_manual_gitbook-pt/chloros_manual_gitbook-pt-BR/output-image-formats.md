# Format Gambar Keluaran

### TIFF (16-bit)

Output format TIFF (.tif) 16-bit (nomor digital piksel 0 - 65535).

### TIFF (32-bit, Persen%)

Keluaran format TIFF (.tif) 32-bit (piksel float 0,0 - 1,0). Nilai piksel mewakili persentase reflektansi dimana 0,0 = 0% dan 1,0 = 100%.

### **JPG (8-bit)**

Output format 8-bit (piksel digital 0 -255) JPG (.jpg).

### **PNG (8-bit)**

Output format 8-bit (piksel digital 0 - 255) PNG (.png). Kompresi lossless cocok untuk tampilan dan visualisasi web.
