# Antarmuka Pengguna Grafis (GUI): Proyek

Chloros memungkinkan Anda untuk membuat proyek yang dapat dibuka kembali di masa mendatang.

## Proyek Baru

<figure><img src=".gitbook/assets/project_new.JPG" alt=""><figcaption></figcaption></figure>

Pilih &quot;Proyek Baru&quot; dari menu utama dan masukkan nama unik untuk proyek Anda.

## Buka Proyek

<figure><img src=".gitbook/assets/project_open.JPG" alt=""><figcaption></figcaption></figure>

Pilih &quot;Buka Proyek&quot; untuk melihat daftar proyek yang ada di Folder Proyek. Jika tidak ada proyek yang ada, menu samping sekunder tidak akan terbuka. Anda dapat melihat beberapa proyek yang dibuat oleh GUI (t1, t2, t3) terdaftar dalam foto di atas. Proyek DATE\_TIME dibuat oleh CLI menggunakan skema penamaan proyek default. Mengklik nama proyek apa pun akan membukanya.

Mengklik tombol &quot;Buka Folder Proyek&quot; akan membuka penjelajah file komputer Anda di jalur proyek. Anda dapat menyesuaikan jalur proyek di [Pengaturan Proyek](project-settings/project-settings.md).
