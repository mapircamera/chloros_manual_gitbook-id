# Format Gambar Keluaran

### TIFF (16-bit)

Format keluaran 16-bit (nomor digital piksel 0 - 65535) dalam format .tif.

### TIFF (32-bit, Persentase %)

32-bit (nilai float piksel 0,0 - 1,0) TIFF (.tif) format output. Nilai piksel mewakili persentase reflektansi, di mana 0,0 = 0% dan 1,0 = 100%.

### **JPG (8-bit)**

8-bit (nilai digital piksel 0 -255) JPG (.jpg) format output.

### **PNG (8-bit)**

8-bit (nilai digital piksel 0 - 255) PNG (.png) format output. Kompresi tanpa kehilangan data yang cocok untuk tampilan web dan visualisasi.
